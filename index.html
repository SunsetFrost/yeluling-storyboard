当前已锁角色与一致性规则

本页优先级不是“花”，而是人物能不能稳定、可复用、可反复补镜。尤其年轻赶尸人必须与阿回同脸，只允许气色与死亡感变化，不允许换成另一张更“帅”的脸。

阿回

最新版脸模基准。瘦长脸、直眉、薄唇、倔，土气但不呆，后续所有阿回镜头以此为准。

年轻赶尸人

必须与阿回同脸，只能更白、更冷、更死，像一张提前走上夜路的阿回，而不是新角色。

老傩

正常老人，不妖道化。干瘦、眼神亮、像真正见过旧事的人，负责给整个故事“口传可信度”。

老赶尸人

作为队伍权威感与仪式感的补充参照，用于后续如需扩拍更完整赶尸系统镜头。

阿回母亲

贫苦山村妇人，压抑、固执、爱意非常笨拙。她不是“哭戏工具人”，而是情感核心。

规则 01
整体坚持湘西民俗恐怖、民国、写实，不仙侠、不古偶、不玄幻滤镜。
规则 02
气质参考：早期张艺谋农村现实质感 +《路边野餐》+ 湘西纪录片，而不是网红古风。
规则 03
道具记忆点固定为：铜铃、黄符、草鞋、米粒、土地庙门槛。后续所有物料都尽量围绕这套意象系统。

16 镜完整版预告片分镜

总时长按 2 分钟控制。前 40 秒给口传民俗与夜路规矩，中段把“遇上了”做实，后段靠同脸关系、母亲喊魂和老傩扣子把恐怖拉成命运感。

01
00:00 — 00:08

录音笔入画，火塘开口

黑场先出铃，再切年轻写书人抬起录音笔的第一视角。火塘边只留老傩与一个模糊陪坐者，空间更像真实采访入口：不是一屋子人围坐听传奇，而是有人专门来问，他也终于肯讲。

镜头功能

建立采访式叙事入口、录音笔记忆点与火塘口传可信度，避免群像分散注意力。

台词 / 音效

VO：“赶尸人，我见过。”
FX：录音笔轻碰掌心，木柴爆裂，极远的铜铃，门外风声。

老傩中近景，旧事像压在眼里

补一个更适合入剧情的老傩镜头，让他不只是一张角色图，而是故事真正的讲述者。火光打在脸沟壑上，像这事他一直没讲完。

镜头功能

台词 / 音效

00:14 — 00:21

母亲塞符

把母子关系提前落地。黄符不是玄学道具，而是母亲笨拙地想把儿子拽回来。这样后面符灰和喊魂才有痛感，不只是功能点。

镜头功能

建立阿回与母亲关系，埋下符纸与“护不住”的命题。

台词 / 音效

VO：“她把符塞进我怀里，说夜路上别回头。”
FX：布料摩擦，手指按进衣襟。

00:21 — 00:28

山路启程

月亮钉在山口，土路一头扎进黑里。预告片从讲述切到亲历，观众被放到阿回脚下那条路上，空间一下变冷。

镜头功能

建立山路尺度与孤绝感，正式入夜。

台词 / 音效

VO：“月亮像钉在山口。”

00:28 — 00:35

阿回夜路特写

这张脸是后面所有反照的基准脸。现已按阿回三视图重新锁脸：阿回得像普通湘西山里少年，怕，但硬撑着走；不是漂亮演员硬演“朴素”。

镜头功能

用阿回三视图锁定基准脸模，为“同脸”反转提供稳定视觉标准。

台词 / 音效

VO：“阿回那时候，还不知道自己撞上了什么。”
FX：呼吸、竹篓轻碰、远处一记铃。

00:35 — 00:42

米粒撒路

补上禁忌系统里最关键的“地面动作”。阿回蹲下撒米，路就从抽象恐怖变成一种民俗里可操作、却也挡不住的规矩。

镜头功能

把“避路 / 请路 / 禁忌提示”做成具体动作，丰富民俗细节。

台词 / 音效

VO：“遇上夜路客，就撒米，让他们认路，莫认人。”
FX：米粒打在泥地上的细碎声。

07
00:42 — 00:49

湿泥里的草鞋印

恐怖第一次真正落地。画面里不该出现草鞋本体，只该留下湿泥里整齐得过分的印痕；观众第一次确认，这东西真的在走，而且已经走到阿回身后。

镜头功能

完成从“听说”到“看见”的第一跳，把恐怖落实为“痕迹”而不是“道具”。

台词 / 音效

VO：“脚印，停在他身后三尺。”
FX：泥水黏连、湿泥拔起声、铃声更近。

08
00:49 — 00:58

赶尸队伍显形

整支预告片第一次完整揭示类型元素。领路的赶尸人必须是斗笠压低、黑纱遮面，像按旧规矩办事的人；队伍整齐、沉默、膝盖不弯，像一支穿过夜山的死队，不要妖魔化到失真。

镜头功能

类型片揭示镜头，确认“赶尸”真实存在，也立住“蒙面是规矩”的民俗质感。

台词 / 音效

VO：“不是一个，是一队。”
FX：铜铃连响，草绳轻撞，黑纱擦过衣领，山风像忽然让开。

09
00:58 — 01:06

赶尸人停铃，说“让”

这是这一版必须补上的节奏铰链。赶尸人一停，铃声断掉，整个世界像被抽空。镜头里只给到斗笠下的黑纱、轮廓和发白的眼神，不把脸完全露出来；只说一个字，比喊叫和怪叫都更瘆人。

镜头功能

制造中段强停顿，把观众和阿回一起钉住，同时保留赶尸人身份信息而不破坏蒙面规矩。

台词 / 音效

对白：“让。”
FX：铃声戛然而止，黑纱轻擦，空气静到能听见心跳。

01:06 — 01:13

黄符烧成灰

母亲给的护符在这里失效。胸口一烫，灰落到衣襟上，观众会明白：规矩知道了也没用，真正的路已经认上你了。

镜头功能

制造一次短促强惊点，同时宣告“护身”失效。

台词 / 音效

VO：“符一热，我就知道坏了。”
FX：纸符冒火、灰扑簌落下。

01:13 — 01:20

土地庙门口草鞋

从山路切到庙门，是空间骤然收紧的一刀。草鞋摆在门槛前，像有人先一步脱下，也像在请人换路，视觉上非常适合做短停顿。

镜头功能

把“庙”与“草鞋”绑定，成为后半段最重要的门槛意象。

台词 / 音效

VO：“庙门口那双鞋，不像给活人穿的。”
FX：门轴轻响，远处铃音穿过门缝。

土地庙内景

神龛、香灰、米粒、黄符全堆在狭窄空间里，民俗感在这里最密。这个镜头不是为了吓，是为了让观众知道“路”和“庙”之间有一套旧规矩在运作。

镜头功能

台词 / 音效

镜头功能

镜头功能

Seedance Prompt 优化规则

这一版分镜按 seedance-prompt-skill 的思路重写：先写技术参数和禁止项，再写角色一致性引用，然后用时间戳分镜法把画面、镜头语言、台词和音效绑在一起，方便直接复制去即梦继续生视频。

规则 A / 技术参数前置
每条 prompt 开头先写画幅比、时长、风格、色调、摄影质感，例如：16:9，24fps，2分钟预告片拆段，湘西民俗恐怖，民国写实，低饱和，电影摄影。
规则 B / 角色引用显式化
把阿回、年轻赶尸人、老傩、母亲写成固定参考位：@图片1 阿回三视图、@图片2 年轻赶尸人三视图、@图片3 老傩、@图片4 母亲，每段镜头都明确引用。
规则 C / 禁止项单列
明确排除：不要仙侠、不要古偶、不要明星脸、不要网红滤镜、不要字幕、不要 Logo、不要现代道具穿帮。
规则 D / 时间戳分镜法
把长视频拆成短段 prompt：例如 0-8秒、9-14秒，每段只负责一个动作和一个情绪，不要把五件事挤在一镜里。
规则 E / 镜头语言写全
必须写景别、角度、运镜、节奏、焦点转移，如：第一视角、缓推、低角度、停顿、突然断铃。
规则 F / 声音与对白并写
Seedance 适合把对白、环境声、音效、情绪直接写进 prompt，尤其《夜路铃》这种强依赖铃声、风声、回声的预告片。

阿回统一写成：瘦长脸、直眉、薄唇、普通湘西山里青年；年轻赶尸人统一写成：与阿回完全同脸，仅肤色更白、眼神更空、更冷、并以斗笠黑纱遮面呈现。
