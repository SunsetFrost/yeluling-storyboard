《夜路铃》
预告片关键帧

一条湘西夜路，从铃响开始。这个页面按 00:00–02:00 的预告片时序展开八个核心镜头，把画面、台词与音效并置成一份可直接讨论的电影式 storyboard 展示页。

时长02:00 Trailer Flow
基调黑底 / 暗红 / 暗金 / 民俗惊悚
用途导演阐述 · 提案展示 · 视觉对时
01
00:00 — 00:12

黑屏 + 铃响 → 火塘讲古

预告片以纯黑切入，先让声音抵达，再让火光浮现。火塘边的讲古感，是整支片子的民俗入口：不是直接吓人，而是先让观众坐下来听一个“夜路上不能回头”的故事。

台词 / 音效

FX：清脆招魂铃声，由远及近。
VO：老人低声开口——“夜路一响铃，活人莫应声。”
Ambience：木柴爆裂、火星轻炸、屋外风声擦过门缝。

02
00:12 — 00:26

山路起点 + 月光

故事正式踏上山路。冷白月光压住山体轮廓，路口像是命运的入口，安静得过分。这个镜头负责把空间感立起来：观众知道，接下来的一切都将发生在一条只能往前、不能回头的路上。

台词 / 音效

FX：山风、虫鸣骤停、布幡轻颤。
VO：青年旁白——“那夜，月亮像钉在山口。”
Music：低频鼓点第一次隐约出现。

03
00:26 — 00:38

草鞋脚印

把视角压低，去看泥地与脚印。草鞋留下的痕迹一深一浅，像有人走过，又像有人被拖着前行。它是“赶尸”这个禁忌意象第一次具体落地，也是预告片的第一层寒意。

台词 / 音效

FX：泥水黏连、鞋底摩擦、铃声节奏更近。
VO：老人——“听见铃，不是赶路的人。”
Music：弦乐拉出一记细长的不安音。

04
00:38 — 00:55

赶尸队伍

队伍终于显形。整齐、沉默、被夜色吞掉一半，像一列从旧传说里走出的影子。这个镜头承担预告片的第一次“揭示”：观众终于确认，夜路上的不只是人。

台词 / 音效

FX：铜铃连响，木杖点地，远处犬吠忽止。
VO：母亲喃喃——“别让他走丢了……”
Music：鼓点加重，低音持续压迫。

05
00:55 — 01:10

土地庙内景

空间由山路转入神龛。香灰、残烛、木像与阴影密不透风，像一个暂时庇护所，也像某种交换正在发生的地方。镜头节奏在这里稍微收紧，让观众有一瞬喘息，又更深地进入禁忌内部。

台词 / 音效

FX：香灰坠落、供桌轻响、门板被风顶开一线。
VO：年轻赶尸人——“过了庙门，就没人能替你回头。”
Ambience：庙内回声短促而空。

06
01:10 — 01:26

年轻赶尸人

主角第一次真正占据画面中心。年轻、克制、带着一种明知不该上路却已经上路的神情。这个镜头给人物立意：不是传统法师式权威，而是一个被命运推进黑夜的人。

台词 / 音效

VO：他低声说——“铃一停，他们就会醒。”
FX：呼吸贴近、指尖摩挲铜铃、风切过衣角。
Music：女声吟唱极轻地浮现一层。

07
01:26 — 01:44

母亲喊魂

情绪在这里抵达峰值。母亲不是旁观者，她用尽气力把“魂”从黑夜里往回喊。这个镜头让惊悚真正刺入人心：所有民俗规矩，最终都落回一个人不肯放手的情感执念。

台词 / 音效

VO：撕裂式喊声——“伢啊，回来——！”
FX：回音在山谷层层退去，铃声被压得尖锐。
Music：弦乐与鼓点同时顶到最高。

08
01:44 — 02:00

主海报 / 收束定格

最后以主海报定格，把预告片里所有意象——夜路、铜铃、山雾、亡者与未尽之念——收拢成一个总视觉记忆点。既像一声落锤，也像铃声还没真正停下。

台词 / 音效

Title Card：《夜路铃》
Tagline：“铃响夜路，莫问归人。”
FX：最后一记孤铃，长尾消失在黑场。
